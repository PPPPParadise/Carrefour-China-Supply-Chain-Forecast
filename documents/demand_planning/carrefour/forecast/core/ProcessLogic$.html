ProcessLogic

Related Doc: package core

object ProcessLogic

This class contains logic for system flow. It contains below steps
1. Load items that may have order today
2. Load item stop period information. For each item load on what days they should not be ordered.
3. Load sales prediction
4. Load current stock level
5. Load DM orders
6. Load past generated orders
7. Submit all data to OrderLogic to calculate order quantity
8. Store order quantity to datalake

此类包含了系统工作逻辑, 有如下步骤
1. 读取所有以系统运行日为订单日的单品
2. 读取单品停止订货信息，确定单品可以和不可以订货的时间段
3. 读取每日销量预测
4. 读取当前库存水平
5. 读取DM订单
6. 读取过去生成的订单
7. 将订单提交到OrderLogic类来计算订单量
8. 将生成的订单量储存在数据池
